# Abstract

Ecogeographical rules such as Bergmann’s and Allen’s have long guided expectations about how organismal size and shape vary across temperature gradients. I synthesize recent empirical literature to evaluate how often Bergmann’s and Allen’s rules are analyzed jointly and assess what is lost when size and shape are studied in isolation. A review of 86 studies published since 2021 reveals that less than half examine both rules together, and far fewer test for co-dependence between them.
